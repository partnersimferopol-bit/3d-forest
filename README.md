# 3Д-лес — подбор подарков

Небольшой статический сайт (без сборки): приветственный лендинг + конструктор подарка + мини-каталог.

## Запуск

- Локально: откройте `index.html` в браузере или `npm start` (нужен Node.js).
- Веб (GitHub Pages): в настройках репозитория включите **Pages → Deploy from branch → main / root**. Сайт: `https://partnersimferopol-bit.github.io/3d-forest/`

Стили подключаются как `assets/styles.css` и `assets/motion.css` — они должны лежать в репозитории рядом с `index.html`.

## Структура

- `index.html` — лендинг + конструктор + каталог + контакты
- `assets/styles.css` — стили (деревянная тема)
- `assets/app.js` — логика: каталог, фильтр бюджета, расчёт, предпросмотр, рекомендации, сохранение/экспорт
- `подбор подарков.html` — старый прототип (оставлен как черновик/история)

## Что менять под себя

- **Контакты**: секция «Контакты» в `index.html`
- **Товары и цены**: массив `PRODUCTS` в `assets/app.js`
- **Логика расчёта**: объект `PRICING` в `assets/app.js`

## Функционал

- расчёт ориентировочной стоимости
- предпросмотр заказа
- 3 рекомендации из мини-каталога
- сохранение шаблона в браузере (LocalStorage)
- копирование заказа в буфер и скачивание `.txt`
