Открыть сайт

На вашем компьютере: «Сохранить на сайте» пишет в localStorage. Откройте главную в том же браузере и обновите Ctrl+F5.

Опубликованный сайт (GitHub Pages): посетители не видят localStorage. После правок нажмите «Скачать JSON» → положите файл в репозиторий как assets/site-content.json (или правьте index.html / assets/catalog.js). Новые фото — в assets/products/, затем git add, git commit, git push. Через 1–2 минуты изменения появятся на partnersimferopol-bit.github.io/3d-forest/.
